## Getting Started

Here follows the instructions to run this project in your pc.

### Prerequisites

* vsCode
* liveServer Extension

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/XDYuuki/pet-city.git
   ```
2. run index.html with liveServer VSCode extension.
